# Customer Segmentation & RFM Analysis (SQL)

## Objective
To segment e-commerce customers using RFM analysis and recommend targeted marketing strategies.

## Dataset
Sample Superstore Dataset (Kaggle)

## Tools Used
- SQL (CTEs, Window Functions, CASE)
- GitHub

## Key Insights
- VIP customers drive majority of revenue
- At-risk customers show churn signals
- Targeted campaigns outperform blanket discounts
